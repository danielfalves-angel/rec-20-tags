HTML

As 20 tags mais importantes

" 1 - HTML "

ela avisa ao navegador que tudo aqui dentro é um HTML

" 2 - HEAD "

nessa ficam informações invisíveis para o usuário

" 3 - BODY "

Tudo o que é visíveis para o usuário

" 4 - TITLE "

define o nome da página e ela fica na tag "head"

" 5 - H1 a H6 "

são os títulos o "H1" é o maior e vai diminuindo até o "H6"

" 6 - P "

é usada para parágrafo para textos

" 7 - BR "

serve para pular uma linha

" 8 - STRONG "

deixa o texto em negrito

" 9 - EM "

deixa o texto em itálico

" 10 - A "

é a tag de link

" 11 - IMG "

Serve para colocar imagens no site

" 12 - UL "

cria uma lista numerada

" 13 - OL "

cria uma lista de bolinhas

" 14 - LI "

é o item da lista se usa em "ol" ou em "ul"

" 15 - DIV "

serve para juntar elementos e organizar o layout.

" 16 - SPAN "

é uma "caixinha" menor, usada para marcar apenas um pedacinho de um texto

" 17 - HEADER "

define o topo do site

" 18 - FOOTER "

define o rodapé do site

" 19 - FORM "

cria um formulário para o usuário colocar dados

" 20 - BUTTON "

cria um botão clicável
